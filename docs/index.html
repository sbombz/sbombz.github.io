🥇 Created Macro.py

Following on from some of my other projects, I wanted to create a general purpose macro program. The program has 2 modes, record and playback. Once record is activated any inputs on mouse or keyboard are logged with a timestamp and location in a .csv file. When the file is ran in playback mode all inputs will be executed in the same location and time. This is useful for automation and allows me to spam check the RSA.ie site for Driving test appointments. (this may seem unethical but there is already hundreds of bots out there doing the same, if you cant beat them join them)

🥇 Switched to Linux Mint

My growing despise for Windows made me try something new. Despite the websites Windows 95 theme, I hate how slow Windows is, I hate how it forces me into updates, I hate Onedrive, I hate Edge and I hate waiting. Switching to Mint was super easy and I still had the option to boot into Windows to play games with kernel-level anticheat. Setting up keybinds made the whole system feel responsive and instantaneous. Without touching my mouse I could open Google Classroom, Firefox, Terminal, Spotify and more in seconds. So glad I made the switch, never going back as long as I can help it
