🥇 Created Macro.py

Following on from some of my other projects, I wanted to create a general purpose macro program. The program has 2 modes, record and playback. Once record is activated any inputs on mouse or keyboard are logged with a timestamp and location in a .csv file. When the file is ran in playback mode all inputs will be executed in the same location and time. This is useful for automation and allows me to spam check the RSA.ie site for Driving test appointments. 🥇 Switched to Linux Mint

My growing despise for Windows made me try something new. I hate how slow Windows is, I hate how it forces me into updates, I hate Onedrive, I hate Edge and I hate waiting. Switching to Mint was super easy and I still had the option to boot into Windows to play games with kernel-level anticheat. So glad I made the switch, never going back as long as I can help it
